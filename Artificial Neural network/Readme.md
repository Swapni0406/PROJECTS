# Hybrid ANN + RAG Copilot System

## Overview
Production-grade AI Copilot combining:
- Artificial Neural Network for intent classification
- Retrieval-Augmented Generation pipeline
- Vector similarity search using FAISS
- FastAPI backend
- Docker containerization
- CI/CD integration

## Key Features
- Modular ML architecture
- ANN-based intent prediction
- Embedding-based semantic retrieval
- RESTful API deployment
- Scalable production-ready design

## Architecture
User Query → ANN → Vector Retrieval → Context Injection → Copilot Response

## Technologies
- Python
- PyTorch
- FAISS
- SentenceTransformers
- FastAPI
- Docker
- GitHub Actions
